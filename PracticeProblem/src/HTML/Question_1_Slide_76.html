Top-Level Class in Java

A top-level class in Java is a class that is declared directly inside a .java file and is not nested inside another class.

A top-level class cannot be declared as private because the private modifier is only meaningful for members inside a class. It restricts access to the enclosing class. But a top-level class has no enclosing class, so private has no meaning here.

Therefore, Java only allows top-level classes to be either public or package-private (default access).

Example

private class MyClass {
}

Error

error: modifier private not allowed here
